--- layout: default title: vTicker ---

  • A jQuery plug-in that adds scrolling
    vertical tickers to your site
  • Just one line of javascript code is
    all you need to start a vTicker
  • Items can be paused when the
    mouse hovers over an item
  • Step backwards and forwards through your
    ticker items with easy client methods

Easy to use: vTicker is easily added to your existing web page by including jQuery, the vTicker plug-in javascript file and then by simply calling the vTicker plug-in with the selector(s) you wish to create a vTicker with:

Example

HTML

{% highlight html %}
  • Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor
  • Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut
  • Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore
{% endhighlight %}

Javascript

{% highlight javascript %} $(function() { $('#example').vTicker(); }); {% endhighlight %}

Result

  • Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor
  • Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut
  • Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore
